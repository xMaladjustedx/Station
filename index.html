About Station To Station

Station to Station is a Vienna club night, that aims to entertain you with an eclectic mix of the best music from various decades.

Here's a selection of the music played at Station to Station:

Bowie // 60s girl groups // (Northern) Soul // Velvet Underground // Patti Smith // Stooges // The Smiths // Joy Division // The Cure // Madchester // Britpop // Future Islands and many more
